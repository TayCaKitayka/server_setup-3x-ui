# ubuntu-server-setup

Автоматические скрипты настройки и защиты сервера.

## Работоспособность

Протестировано на Ubuntu 22.04 LTS

## Скрипты

| Скрипт | Описание |
|--------|----------|
| `setup.sh` | Базовая настройка — SSH, UFW, блокировка ping, отключение IPv6 |
| `setup-3xui.sh` | Полная настройка — всё выше + установка 3x-ui + BBR |

## Возможности

- 🔒 Вход по SSH только по ключу (пароль отключён)
- 🔢 Кастомный порт SSH
- 🛡️ Настройка файрвола UFW
- 🚫 Блокировка ICMP ping навсегда (iptables + sysctl)
- 🚫 Отключение IPv6 навсегда (sysctl, без GRUB)
- 📦 Установка и автонастройка 3x-ui (только setup-3xui.sh)
- ⚡ Включение BBR (только setup-3xui.sh)
- 🤖 Полностью автоматический запуск — ничего вводить не нужно

## Требования

- Ubuntu 22.04 LTS
- Доступ от root

## Использование

### 1. Заполните конфигурацию

Откройте скрипт и заполните блок конфигурации в начале файла:

```bash
# Для обоих скриптов:
SSH_PORT=22663                    # Порт SSH
SSH_PUB_KEY="ВАШ_ПУБЛИЧНЫЙ_КЛЮЧ" # Публичный ключ (ssh-ed25519 ...)

# Только для setup-3xui.sh:
XUI_PORT=34821                    # Порт панели 3x-ui
XUI_USER="ВАШ_ЛОГИН"              # Логин панели
XUI_PASS="ВАШ_ПАРОЛЬ"             # Пароль панели
```
### 2. Загрузите и запустите

```bash
# Загрузите скрипт на сервер
scp setup.sh root@IP_ВАШЕГО_СЕРВЕРА:~/

# Подключитесь и запустите
ssh root@IP_ВАШЕГО_СЕРВЕРА
chmod +x setup.sh
./setup.sh
```

### 3. Проверьте SSH (не закрывая текущую сессию!)
```bash
ssh -p 22663 -i ~/.ssh/id_ed25519 root@IP_ВАШЕГО_СЕРВЕРА
```


Открытые порты по умолчанию

22663 - TCP - SSH
80 - TCP - HTTP
443 - TCP - HTTPS
8443 - TCP - HTTPS
34821 - TCP - 3x-ui Panel (only for setup-3xui.sh)

⚠️ Важно
Не закрывайте текущую SSH сессию после запуска скрипта,
пока не убедитесь что можете подключиться на новом порту SSH.
Иначе вы потеряете доступ к серверу.
